# Claude Config Module

Basic Claude Code configuration and commands for AI Code Forge.

## Purpose

This module provides the foundational Claude Code configuration including:
- Essential Claude Code settings
- Core command definitions
- Foundation and specialist agents
- Status line configuration

## What's Included

- **Settings**: Basic Claude Code configuration without hardcoded model settings
- **Commands**: Essential slash commands for development workflow
- **Agents**: Foundation agents (context, patterns, principles, critic, conflicts, researcher) and specialist agents (git-workflow, github-pr-workflow, etc.)

## Installation

```bash
acforge module install claude-config
```

## Configuration

The module installs Claude Code configuration without specifying a model, allowing users to:
- Use their preferred Claude model
- Configure model settings in their Claude Code preferences
- Maintain personal model preferences

## Dependencies

None. This is a foundation module required by most other modules.

## Compatibility

- Claude Code: >=2.78.0
- AI Code Forge: >=4.0.0
